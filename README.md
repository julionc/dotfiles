# dotfiles

My config files for fish, git and so on

## Shell

### Fish

```properties
brew install fish
```

You will need to add:

```properties
/usr/local/bin/fish
```

to /etc/shells.

To make fish your default shell. Run:

```properties
chsh -s /usr/local/bin/fish
```
